# my .dotfiles

These scripts and configuration files automatically build my personal tooling on a fresh MacOS installation.

### Structure

```bash
├── .vscode
│   └── settings.json
├── README.md
├── install.core.sh
├── packages
│   ├── Brewfile
│   ├── git/
│   ├── install.packages.sh
│   ├── neovim/
│   ├── oh.my.zsh/
│   └── pyenv/
│   
├── runcom
│   └── .bash_profile
└── system
    ├── .alias
    ├── .env
    ├── .function
    ├── .path
    └── .prompt
```

### Installation

```bash
bash install.core.sh
```
